# MERN Stack Todo App

A full-stack Todo application built using the MERN stack (MongoDB, Express, React, Node.js).
Users can create, update, complete, and delete tasks in real time.

## 🚀 Live Demo

Frontend (Vercel)
https://todo-lime-two.vercel.app

Backend API (Render)
https://todo-vh4w.onrender.com/api/v1/todos

## ⚙️ Environment Variables

Create a `.env` file inside backend folder.

```
PORT=3000
YOUR_CONNECTION_STRING=your_mongodb_connection_string
```

---

## 🧪 Run Locally

Clone the project

```
git clone https://github.com/abhi281500/todo
```

Install dependencies

```
cd backend
npm install
```

Run backend

```
node server.js
```

Run frontend

```
cd frontend
npm install
npm run dev
```
